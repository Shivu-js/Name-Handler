# 📝 Listy

A simple and fast **Todo + Lists PWA** built with JavaScript.

Listy helps you manage tasks and lists easily, even **offline**.

---

## ✨ Features

- ✅ Tasks management
- 📂 Lists support
- ⚡ Works offline (PWA)
- 💾 Saves data with LocalStorage
- 📥 Export list to TXT
- 🎨 Clean minimal UI

---

## 🚀 Demo

Try Listy here:  
[Listy](https://shivu-js.github.io/Listy)

---

## 📱 Install as App

1. Open Listy in your browser
2. Click **Install App**
3. Use it like a normal mobile app
